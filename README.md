# Connor & Laura Wedding Site

A single-page wedding invitation built with Vue 3 (Composition API) and TypeScript, designed for GitHub Pages hosting.

## Commands

- `npm install` - install dependencies
- `npm run dev` - start local dev server
- `npm run build` - generate production build
- `npm run preview` - preview the production build locally

## Customization

- Replace placeholder gallery images in `public/photos` with your engagement photos (keep the same file names or update the paths in `src/App.vue`).
- Update the RSVP URL (`rsvpLink` in `src/App.vue`) to your Google Form.
- Edit schedule/location text in `src/App.vue` to match your plans.
